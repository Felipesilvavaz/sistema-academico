# Sistema Acadêmico v1.1 - Módulo Cadastro de Alunos

Projeto desenvolvido para a disciplina de Gerência de Configuração e Controle de Versão — UniFil.

## Descrição

Sistema web acadêmico para cadastro e gerenciamento de alunos.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Como executar

1. Clone o repositório:
   git clone https://github.com/Felipesilvavaz/sistema-academico.git

2. Abra o arquivo index.html no navegador.
